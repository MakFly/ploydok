# Project Docs

Internal project documentation for Ploydok. Public, user-facing docs now live in
the standalone docs preview repository (`~/Documents/lab/tools/docs` locally).

## Index

- `roadmap/` — sprint roadmap, scope, DoD, and release tracking.
- `decisions/` — architecture and product decisions.
- `plans/` — implementation plans and archived sprint plans.
- `operations/` — install notes, runbooks, and security procedures.
- `research/` — benchmark notes, gap analysis, and test strategy.
- `audits/` — dated product and UX audits.
- `fixtures/` — reference fixtures used by research or tests.
