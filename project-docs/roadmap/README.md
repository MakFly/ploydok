## Légende des statuts

- **✅ Terminé** — code mergé **et** DoD validée bout-en-bout (test vert, endpoint audité, e2e green). Vit dans `done/`.
- **✅ Code · ⏳ e2e** — code mergé, tests unit verts, mais e2e Playwright pas encore exécutée. Vit dans `wip/`.
- **⚠️ Partiel** — scope partiellement implémenté, certains items DoD pas livrés. Vit dans `wip/`.
- **⏳ À faire** — pas commencé ou en cours d'implémentation. Vit dans `wip/`.
